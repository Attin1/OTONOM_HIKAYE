Kullanıcı Tercihleri ve Duygu Analizi Temelli Doğrusal Olmayan Otonom Hikâye Üretim Sistemi

Bu proje, TÜBİTAK 2204-A Lise Öğrencileri Araştırma Projeleri Yarışması kapsamında geliştirilmiştir.

Bu çalışmanın amacı, geleneksel doğrusal anlatı yapılarının sınırlarını aşarak kullanıcı tercihleri ve duygu durumuna göre dinamik biçimde şekillenen otonom bir hikâye üretim sistemi ortaya koymaktır.

Proje Özeti

Geleneksel hikâye anlatımı çoğu zaman sabit ve değişmez bir yapı sunar. Bu projede anlatı; kullanıcıdan alınan seçimler ve metin tabanlı duygu analizi sonuçlarına bağlı olarak sürekli yeniden yapılandırılır. Böylece her kullanıcı için benzersiz, doğrusal olmayan ve etkileşimli bir hikâye deneyimi oluşur.

Sistem, kullanıcı etkileşimlerini analiz eder, hikâye akışını duygusal bağlama göre günceller ve önceden belirlenmiş senaryolara bağlı kalmadan anlatıyı otonom biçimde sürdürür.

Temel Özellikler

Doğrusal olmayan (non-linear) hikâye yapısı
Metin tabanlı duygu analizi
Kullanıcı tercihlerine duyarlı anlatı dallanması
Otonom hikâye üretim mekanizması
Her etkileşimde değişebilen dinamik anlatı akışı

Projenin Amacı

Bu çalışmanın temel hedefi, dijital çağın etkileşim odaklı anlatı beklentilerine yanıt verebilen, kullanıcıyı pasif bir okuyucu olmaktan çıkararak anlatının aktif bir bileşeni hâline getiren bir sistem geliştirmektir.



Non-Linear Autonomous Story Generation System Based on User Preferences and Emotion Analysis

This project was developed for the TÜBİTAK 2204-A High School Students Research Projects Competition.

Project Overview

Traditional storytelling structures often follow a linear and static narrative. This project aims to overcome these limitations by introducing an autonomous story generation system that dynamically adapts the narrative flow based on user choices and emotion analysis of user inputs.

The system continuously reshapes the story without relying on fixed scenarios, providing a unique and interactive narrative experience for each user.

Key Features

Non-linear narrative structure
Text-based emotion analysis
User preference–driven story branching
Autonomous narrative generation
Dynamic and evolving story flow

Objective

The main objective of this project is to design an interactive storytelling system that transforms the user from a passive reader into an active participant, enabling personalized and emotionally adaptive narratives.
